# Крафтерра

**Крафтерра** — онлайн-витрина крафтовых товаров, локальных брендов и малых производителей.

Платформа помогает пасечникам, мыловарам, рукодельницам, чайным мастерам, фермерам, кондитерам, мастерам свечей, декора, косметики и другим малым брендам красиво представить свои товары онлайн и получать заявки от покупателей.

## Важная модель проекта

Крафтерра — это не доска объявлений и не классический маркетплейс с оплатой через платформу.

Платформа не занимается:

- оплатой заказов между покупателем и продавцом;
- упаковкой товаров;
- хранением товаров;
- доставкой товаров;
- комиссией с заказов.

Все вопросы оплаты, упаковки, доставки, наличия и выполнения заказа остаются на стороне продавца.

## Что делает Крафтерра

Для продавцов:

- страница бренда;
- карточки товаров;
- каталог товаров;
- заявки от покупателей;
- отзывы;
- базовая статистика;
- платное продвижение внутри платформы.

Для покупателей:

- поиск крафтовых товаров;
- подборки по категориям и поводам;
- страницы локальных брендов;
- контакты продавцов;
- возможность оставить заявку или перейти к продавцу.

## Монетизация

### Бесплатный период

Первые 30 дней бесплатно.

### Тарифы для продавцов

- **Старт** — 390₽/мес
- **База** — 990₽/мес
- **Бренд** — 1990₽/мес

### Платное продвижение

- место в подборке;
- товар в топе категории;
- баннер на главной;
- участие в сезонной коллекции;
- раздел «Бренд недели»;
- продвижение к праздникам.

## Позиционирование

**Крафтерра — территория крафтовых товаров и локальных брендов.**

Миссия проекта — дать малым производителям простой онлайн-инструмент, который помогает им стать заметнее, красиво представить продукт и получать заявки от покупателей.

## Технический старт

Проект планируется как web-приложение на React + TypeScript.

Первая версия должна включать:

- главную страницу;
- каталог брендов;
- каталог товаров;
- страницу бренда;
- страницу товара;
- тарифы для продавцов;
- рекламные места и продвижение;
- форму заявки продавца;
- форму связи покупателя с продавцом.
